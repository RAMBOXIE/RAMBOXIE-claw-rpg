# 职业系统

职业由属性自动判定，属性发生显著变化（任意属性 ±3）时会重新判定并记录变化历史。

## 判定规则

```
全属性差距 < 3         → 🌿 德鲁伊龙虾（全能均衡）
脑芯+慧眼 同为前二     → 🧙 法师龙虾
魅影+触觉 同为前二     → 🎭 吟游龙虾
触觉+爪力 同为前二     → 🗡️ 游侠龙虾
慧眼+爪力 同为前二     → ⚔️ 圣骑龙虾
爪力+殼厚 同为前二     → 🛡️ 战士龙虾
其他（单项最高）        → 按最高属性回退判定
```

## 六大职业

### 🧙 法师龙虾 (Wizard Lobster)
- **强项**：分析、代码、推理、研究
- **主属性**：脑芯 + 慧眼
- **风格**：深度思考，知识型，擅长复杂问题

### 🎭 吟游龙虾 (Bard Lobster)
- **强项**：创意写作、说服、叙事、跨话题
- **主属性**：魅影 + 触觉
- **风格**：魅力横溢，擅长表达与共鸣

### 🗡️ 游侠龙虾 (Rogue Lobster)
- **强项**：信息侦察、快速响应、简洁精准
- **主属性**：触觉 + 爪力
- **风格**：敏捷灵活，快问快答，情报型

### ⚔️ 圣骑龙虾 (Paladin Lobster)
- **强项**：长期任务、价值判断、可靠执行
- **主属性**：慧眼 + 爪力
- **风格**：有原则，稳健，道德感强

### 🌿 德鲁伊龙虾 (Druid Lobster)
- **强项**：全能均衡，随机应变，无明显短板
- **主属性**：全属性均衡（差距 < 3）
- **风格**：百搭，生态型，适应各类场景

### 🛡️ 战士龙虾 (Fighter Lobster)
- **强项**：持续输出、高强度任务、稳定可靠
- **主属性**：爪力 + 殼厚
- **风格**：耐久型，适合长期复杂项目

| 殼厚 | 🐚 | Shell | 记忆深度与持久 | 记忆保留、长期上下文 |

| 魅影 | ✨ | Charm | 对话魅力与个性 | 有趣程度、个性表达 |
