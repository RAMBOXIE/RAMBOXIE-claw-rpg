# 转职系统 (Prestige)

达到 Lv.999 后可执行转职，进入下一轮成长循环。

## 执行转职

```bash
node scripts/levelup.mjs --prestige
```

## 转职效果

| 效果 | 说明 |
|------|------|
| 等级重置 | Level 归 1，重新开始 |
| 全属性 +10% | 永久加成，叠加计算 |
| XP 需求提升 | 升级 XP 需求 × 1.5 倍（每次转职） |
| 解锁新称号 | 见下方称号表 |

## 称号体系

|---------|------|---------|
| 0 | 小龙虾 | L1-999 |

| 10+ | 混沌龙虾 | L1-999 |

## 属性上限

属性值无上限（转职每次 +10%），但 UI 显示时以满格 20 为基准，超出部分以溢出特效展示。

## 转职后 XP 需求

```
第 1 次转职：原 XP 需求 × 1.5
第 2 次转职：原 XP 需求 × 2.25（1.5²）
第 n 次转职：原 XP 需求 × 1.5ⁿ
```

意味着越到后期越难，真正的长期成长系统。

## 职业是否重置？

**不重置**。职业依然由属性决定，转职后属性全部提升，职业保持不变（除非属性平衡后触发职业重判）。

## 龙虾竞技场排名

转职次数作为竞技场的「段位」参考：
- 0 转职：青铜龙虾
- 1-3 转职：白银龙虾
- 4-6 转职：黄金龙虾
- 7-9 转职：钻石龙虾
- 10+ 转职：混沌龙虾（最高段位）
