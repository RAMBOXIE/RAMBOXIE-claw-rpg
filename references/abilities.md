# Abilities

Abilities unlock automatically as you level up, distributed by class.

## Unlock Thresholds

```
Lv.1  → Starter ability (class foundation)
Lv.5  → Advanced ability
Lv.10 → Mastery ability
Lv.20 → Legendary ability
```
